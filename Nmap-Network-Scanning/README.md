# Nmap Network Scanning Lab

## Objective
Perform a network scan using Nmap to identify open ports, running services, and potential security risks on a target system.

## Tools Used
- Nmap
- Kali Linux (or your OS)
- Test target (VM or local machine)

## Commands Executed
